# Web Tasarım Final Projesi

Bu proje, **Bilgisayar Programcılığı 1. sınıf 1. dönem** kapsamında alınan **Web Tasarım** dersi için hazırlanmış **final projesidir**.  
Proje sürecinde amaç, temel web tasarım mantığını öğrenmek ve sadece HTML ve CSS kullanarak sıfırdan bir web sitesi geliştirmektir.

## Proje Hakkında

- Bölüm: Bilgisayar Programcılığı  
- Sınıf / Dönem: 1. Sınıf / 1. Dönem  
- Ders: Web Tasarım  
- Proje Türü: Final Projesi  
- Alınan Not: **AA (100/100)**  
- Akademik Değerlendirme:  
  - Görevli öğretmen tarafından **örnek 100’lük proje** olarak seçilmiştir.

## Kullanılan Teknolojiler

- HTML5  
- CSS3  

Proje geliştirilirken:
- JavaScript kullanılmamıştır  
- Herhangi bir framework, hazır tema veya kütüphane kullanılmamıştır  
- Tüm yapı ve tasarım elle kodlanmıştır

## Yapay Zekâ Kullanımı Hakkında

Bu proje bir **yapay zekâ ürünü değildir**.  
Geliştirme sürecinde yalnızca öğrenme amacıyla (konu araştırma, hata kontrolü gibi) yapay zekâdan sınırlı şekilde destek alınmıştır.  
Kod yapısı, tasarım kararları ve sayfa düzeni **tamamen öğrenci tarafından** oluşturulmuştur.

## Projenin Amacı

Bu proje ile:
- HTML etiketlerinin doğru ve semantik kullanımını öğrenmek  
- CSS ile sayfa düzeni, renk uyumu ve temel tasarım prensiplerini uygulamak  
- Framework kullanmadan web sitesi geliştirme mantığını kavramak  
amaçlanmıştır.
